# Test Task
## Installation

1. Clone project from GitHub
``` bash
git clone git@github.com:IlyaHar/TestTask.git 
```

## Settings

2. **Copy all from .env.example, create .env and paste it there. Configure database.**

3. Install Composer and Npm
``` bash
 composer install
```
``` bash
 npm install
```
``` bash
 php artisan key:generate
```
``` bash
 php artisan migrate
```
``` bash
 npm run dev
```
``` bash
 php artisan serve
```
